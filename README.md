# Gangster Sim

A prohibition-era gangster simulation game. Mobile-first, vanilla JS.

## Play

Double-click `index.html` to open in your browser — no server required.

Or serve locally if you prefer:

```bash
python -m http.server 3456
```

## Gameplay

- Tap locations on the map to travel
- Talk to characters, take jobs, mug marks, extort businesses
- Buy weapons from Pete, vehicles from Gus, medicine at the pharmacy
- Collect extortion payments weekly — miss collections and agreements fall apart
- Watch your relationships — no visible meters, but people remember

## Save

Game state auto-saves to `localStorage` after every action.

## Stack

- Vanilla HTML/CSS/JS (classic scripts, no build step)
